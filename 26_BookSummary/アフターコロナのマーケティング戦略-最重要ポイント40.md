## フレームワーク詳細

### 論点1への回答: 連続的変化の認識

**定義**:
コロナ後に目立った変化も、実はこれまでずっと顧客心理・行動の変化の延長線上にある

**構成要素**:

1. **小さい変化を早く読み取る**
   - 昨日の延長線上に今日、今日の延長線上に明日があると思わない
   - 顧客心理・行動の変化をリアルタイムで感じる

2. **変化に対応し続ける**
   - 異なる戦略シナリオを検証し続ける
   - なんとなく今日は昨日の延長であるという前提を問題視

3. **複数シナリオの準備**
   - 何か起こったらすぐに動けるように準備
   - 1つの戦略で固定しない

**事例**:
- **スターバックス**: ピックアップ型テストは数年前から実施。コロナで場当たり的ではない
- **マクドナルド**: 2017年発表の5つの戦略（4つ目デジタル、5つ目デリバリー）の実行。パニック的動きではない

**前提条件/注意点**:
- 常に進行している変化が、イベント発生で一気に加速する可能性
- ビジネスで1つの戦略・シナリオで固定するのは重大なリスク

### マス思考と平均化の罠

**定義**:
顧客を画一的に捉え、平均値で考えることによる重大な誤り

**問題点**:

1. **パーセンテージだけ見る罠**
   - 10-20代の支持が高い → でも人口は50-60代の1.5倍少ない
   - 50-60代の比率は低い → でも人口で掛け算すると市場は大きい

2. **単一メッセージの罠**
   - マス媒体で単一訴求は非効率
   - すべてデジタルにすると多くの潜在顧客を逃す

3. **世代間差異の無視**
   - 世代間の差が大きい
   - 地方と都市部でも状況が異なる

**解決策**:
- 複数の顧客セグメントに、どういうメッセージを、どういうメディアで伝えるか
- かなり複雑な戦略、メディア、顧客、メッセージの組み合わせが必要

**キーポイント**:
デジタルかマス媒体かが問題ではなく、顧客特性を考えずに単一のメッセージ訴求でよしとする「マス思考」こそが問題

### パラレルワールド論

**定義**:
昭和世代と令和世代は、同じ日本に住みながらまったく異なる世界に住んでいる

**2つの世界**:

| 項目 | 昭和世代（50代以上） | 令和世代（20-30代） |
|------|---------------------|-------------------|
| メディア利用 | テレビ中心 | デジタル中心 |
| 価値観形成 | 昭和の成功体験 | デジタル環境下 |
| 連絡手段 | 固定電話、対面 | スマホ、オンライン |
| 支払い | 現金 | デジタル決済 |
| 地図 | 紙の地図 | スマホアプリ |

**問題**:
- 見た目も似ていて、言語も同じなので、その差に気がつかない
- お互いが実はまったく違う世界に住んでいることに気づかない
- 職場で大きなすれ違いが生じる

**解決策**:
- まったく違う国で生まれ育ったくらいの差異があると自己認識
- ターゲット顧客は誰なのか？
- その顧客を一番知っているのは誰なのか？
- その顧客にとって何が重要なのか？

**事例**:
- ケニアのマーケット参入と同じレベルの差異
- 現地の優秀な人を採用、現地に詳しい人に任せる
- 同様に、世代間でも互いの知見を活かすべき

### テレビとデジタルメディアの特性比較

**定義**:
テレビとデジタルメディアはまったく別物であり、それぞれ異なる特性を持つ

**テレビの特性**:

| 項目 | 詳細 |
|------|------|
| カバレッジ | 短時間で何百万人に一瞬でリーチ |
| 視聴率 | 5%でも何百万人単位 |
| 訴求力 | 視聴1日後の想起率: 悪くても10%程度、よければ80%程度 |
| 態度 | 受動的に情報を受け取る |
| 認知形成 | 一気に影響力を及ぼせる |
| コスト | 1人あたりリーチコストが最も安い |
| ターゲティング | 緩い、直接コンバージョンへのコスト効率は悪い |

**デジタルメディアの特性**:

| 項目 | 詳細 |
|------|------|
| カバレッジ | 無数に細分化、短時間カバレッジ速度は遅い |
| リーチ | ツイッター等のユーザー数は多いが接触は短時間・たまたま |
| 訴求力 | 動画CM・バナー・リスティングの視認1日後想起率: 非常に低い、多くはゼロ |
| 態度 | 積極的、意識的、取捨選択 |
| パーソナライゼーション | 細かく設定可能 |
| 効果計測 | 個々の配信効果を計測可能、費用対効果管理・改善しやすい |
| 刈り取り | コンバージョン面での寄与あり |

**データ**:
- 視聴率データ: 50代以上テレビ、30-40代拮抗、30代以下デジタル優勢
- 15-20年前は明確に分かれていなかったが、現状は世代差が非常に顕著
- 広告費: テレビは2000年前後ピーク、現状は1990年代と変わらず
- 落ち込みが厳しいのは雑誌と新聞
- インターネットメディアだけが1990年代後半から異常に伸びている

**結論**:
テレビであろうとデジタルメディアであろうと、顧客起点でターゲットごとにリーチ効率と異なる。すべての顧客ターゲットに最適なメディアミックスを、マスとデジタルを俯瞰して考えるべき

### WHO, WHAT, HOWの順序

**定義**:
マーケティング戦略を考える際の議論順序

**正しい順序**:

1. **WHO（誰に）**
   - ターゲット顧客の特定
   - 顧客の解像度を高める

2. **WHAT（何を）**
   - どんな価値を提供するか
   - 顧客のニーズに合わせた便益

3. **HOW（どうやって）**
   - どう伝えるか、どう届けるか
   - メディアミックス、4P

**誤った順序**:
HOWから始める → 手段先行で顧客不在

**前提条件/注意点**:
- 顧客を起点にする
- 顧客の変化をリアルタイムで捉える

### 9セグマップ（顧客の解像度を高める）

**定義**:
顧客を9つのセグメントに分類し、それぞれの特性と優先度を明確にするフレームワーク

**軸**:
- 縦軸: ブランドに対する態度（ロイヤル、スイッチャー、拒否層）
- 横軸: カテゴリーに対する態度（ヘビー、ミディアム、ライト）

**9セグメント**:

| カテゴリー/ブランド | ヘビーユーザー | ミディアムユーザー | ライトユーザー |
|-------------------|--------------|-----------------|--------------|
| ロイヤル顧客 | ロイヤル・ヘビー | ロイヤル・ミディアム | ロイヤル・ライト |
| スイッチャー | スイッチャー・ヘビー | スイッチャー・ミディアム | スイッチャー・ライト |
| 拒否層 | 拒否・ヘビー | 拒否・ミディアム | 拒否・ライト |

**活用方法**:
1. 各セグメントの人数と売上貢献を把握
2. 優先順位を決定
3. セグメントごとに異なる戦略を立てる

**よくある誤解**:
- ロイヤル顧客だけに注力 → 間違い
- ロイヤル顧客の要望に応える → 必ずしも正しくない
- ブランドイメージを守るのはロイヤル顧客のため → 間違い

**正解**:
- ブランドイメージは非ユーザーや拒否層のためにも重要
- スイッチャーの獲得が売上拡大の鍵
- カテゴリー自体の拡大も視野に

### 論理ではモノは売れない

**定義**:
人間は感情で買って、後から理屈で正当化する

**構成要素**:

1. **感情が先、理屈は後**
   - 購買の瞬間は感情
   - 購買後に理屈で正当化

2. **便益と独自性を紐づける**
   - 論理的な便益だけでは不十分
   - 感情に訴える独自性が必要

3. **伝え方の重要性**
   - 何を伝えるかより、どう伝えるかが重要
   - まず製品・サービスをつくり、後から伝え方を考えるのは間違い

**事例の示唆**:
- ユニクロの夏マスク: 機能訴求だけでは不十分
- カップヌードル: 同じパターンの広告でも常に訴求力がある理由

---

### ポートフォリオ経営

**定義**:
1つの事業や提供方法だけに集中せず、複数の対応がとれるように準備すること

**構成要素**:

1. **複数シナリオの準備**
   - 顧客の心理や行動変化を常に意識
   - 何か起こったらすぐに動けるように準備

2. **柔軟な転換**
   - 別の方向性に変化しても柔軟に転換できる状態
   - 複数の対応可能

3. **平時の備え**
   - 有事の選択スピードは平時の備えから生じる
   - ポートフォリオが持てない場合は機敏な動きでカバー

**事例**:
- スターバックス: ピックアップ型を数年前からテスト
- マクドナルド: 2017年から戦略準備

**キーポイント**:
変化に対する複数のシナリオを持つことが、アフターコロナだけでなく、常に求められている

## データ・研究結果

### メディア利用時間（世代別）

| 世代 | テレビ | インターネット |
|------|-------|--------------|
| 50代以上 | 優勢 | - |
| 30-40代 | 拮抗 | 拮抗 |
| 30代以下 | - | かなり優勢 |

**注**: 15-20年前は明確に分かれていなかった

### 世代人口の違い

| 世代 | 同年生まれ人口 |
|------|--------------|
| 50代 | 200万人以上 |
| 現在の子どもたち | 90万人程度 |

**比率**: 50代以上と10代・20代では、世代人口が約1.5倍違う

### 広告費の推移

| メディア | 傾向 |
|---------|------|
| テレビ | 2000年前後ピーク、現状は1990年代と変わらず |
| 雑誌・新聞 | 落ち込みが厳しい |
| ラジオ | 多少落ちている |
| インターネット | 1990年代後半から異常に伸びている |

**インターネット広告**: 2019年にテレビ広告費を抜いた（ただし誤解あり）

### 外出率の推移

| 年 | 1日の移動回数 | 外出率 |
|----|-------------|-------|
| 1987年 | 2.63回 | 86.3% |
| 2015年 | 2.17回 | 80.7% |

**出典**: 国土交通省「都市における人の動きとその変化」（5年ごと公表）

**結論**: 長期的トレンドとして、日本人は外出や移動をあまりしなくなっていた

### 視聴態度と想起率

| メディア | 視聴1日後の想起率 |
|---------|----------------|
| テレビCM | 悪くても10%程度、よければ80%程度 |
| デジタルメディア（動画CM・バナー・リスティング） | 非常に低い、多くの場合ゼロ |

**結論**: 一気に影響力を及ぼせるという点では断然テレビ。デジタルは「刈り取り向け」

## 引用・参考

### 論点構成（40の論点）

**第1章: デジタル時代のマーケティングの誤解**
- 論点2: デジタル時代にマスマーケティングは不要だ
- 論点3: テレビなどの旧メディアはオワコンだ
- 論点4: デジタルマーケティングはデジタルのことがわかる人に任せておけばいい
- 論点5: これからのマーケティング活動は、デジタルネイティブが圧倒的に有利だ
- 論点6: リアルでは限界があるから、今後はネット主体にシフトすべき
- 論点7: ビッグデータを持てば、最適なマーケティングが可能になる

**第2章: 顧客理解における誤解**
- 論点8: 顧客の声を聞いて既存製品の改良や新製品の開発をすれば、売れるものができる
- 論点9: 消費者調査は大いなる無駄である
- 論点10: カスタマージャーニーを描けば、いい打ち手が見つかる
- 論点11: 優れた製品やサービスを提供すれば、自然に売れるようになる
- 論点12: ロイヤル顧客の要望に応えることが、いいマーケティング活動だ
- 論点13: B2BとB2Cは別物だから、同じマーケティング理論は使えない

**第3章: ブランディングの誤解**
- 論点14: ブランド名が認知され、ブランドイメージがよくなれば、よく売れる
- 論点15: ブランド価値を高めるには、洗練されたデザインが重要である
- 論点16: ブランドイメージを守るのは、ロイヤル顧客のため
- 論点17: SDGsを意識した施策を打てば、企業イメージも売上も向上する
- 論点18: ブランドイメージは一貫性が大切だから、変えてはいけない

**第4章: プロモーションの誤解**
- 論点19: プロモーションの役割は、認知度を上げて売上を伸ばすことだ
- 論点20: トレンドには常に乗っかるべき
- 論点21: 人気タレントを起用して注目されれば、ヒットになる
- 論点22: SNSでバズれば、売上が伸びる
- 論点23: 有名クリエイターに頼めば、いい広告やキャンペーンができる
- 論点24: 同じパターンの広告を続けるのは、マーケターの怠慢である
- 論点25: まずは製品やサービスをつくり、それから伝え方を考えればいい
- 論点26: 差別化できる要素がなければ、ヒットはつくれない

**第5章: 戦略策定の誤解**
- 論点27: 論理的に考えて戦略を策定すれば、事業は成長する
- 論点28: 戦略を策定したら、あとはそれを実行するだけだ
- 論点29: マーケティング戦略とは、4Pを考えることだ

## 実践ポイント

### 顧客起点で考える

1. **WHO（誰に）を最優先**
   - ターゲット顧客の特定
   - 顧客の解像度を高める（9セグマップ等）

2. **顧客を画一的に捉えない**
   - マス思考の罠を避ける
   - 1人1人の顧客心理に寄り添う

3. **パーセンテージだけでなく人口も見る**
   - 50-60代の人口は10-20代の1.5倍
   - 市場規模を正しく把握

### 世代間差異を認識する

1. **パラレルワールドを理解**
   - 昭和世代と令和世代は全く異なる世界
   - 見た目や言語が同じでも中身は別物

2. **ケニア参入と同じレベルの差異**
   - 現地の優秀な人を採用
   - 互いの知見を活かす

3. **デジタルネイティブも万能ではない**
   - 「デジタルを見ていない」顧客層の存在
   - 情報格差は逆に広がっている

### メディアミックスを最適化

1. **テレビとデジタルを統合的に考える**
   - どちらかではなく、組み合わせ
   - 顧客起点で最適なメディアミックス

2. **それぞれの特性を理解**
   - テレビ: 短時間で大量リーチ、認知形成
   - デジタル: パーソナライゼーション、刈り取り

3. **顧客から見た一貫性**
   - 社内事情は顧客に関係ない
   - 全メディアで一貫したメッセージ

### 複数シナリオで備える

1. **1つの戦略に固定しない**
   - 変化に対する複数シナリオ準備
   - 何か起こったらすぐ動ける

2. **ポートフォリオ経営**
   - 1つの事業・方法に集中しない
   - 柔軟に転換できる状態

3. **平時の備えが重要**
   - 有事の選択スピードは平時から
   - 数年前からテスト・準備

### 顧客の変化をリアルタイムで捉える

1. **昨日の延長線上に今日はない**
   - 常に変化し続けると認識
   - 小さい変化を早く読み取る

2. **外出率の長期トレンド**
   - 1987年→2015年で移動回数・外出率減少
   - コロナ以前から起こっていた変化

3. **コロナは加速装置**
   - 非連続に見えるが、実は連続的変化の加速
   - 以前からの変化を認識していれば対応可能だった
