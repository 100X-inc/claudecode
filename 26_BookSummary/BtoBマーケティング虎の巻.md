## フレームワーク詳細

### STP（セグメンテーション・ターゲティング・ポジショニング）

**定義**:
マーケティング戦略の基本。すべての顧客層を狙うのではなく、最も自社サービスが刺さる層に絞り込む。

**構成要素**:
1. **セグメンテーション**: 業種、売上規模、従業員数、部署、役職、性別、抱える課題で分類
2. **ターゲティング**: 一番刺さる可能性が高い層を選択
3. **ポジショニング**: 競合との差別化ポイントを明確化

**適用方法**:
- 想定顧客層を洗い出す
- 各層の特性と自社適合度を評価
- 1つの層に集中してリソース投下
- 外れた場合は次の層へ高速移行

**前提条件/注意点**:
「みんなに好かれようとして、みんなに嫌われる」(仲畑貴志)。全顧客層へのアプローチは非効率。

### 売上の因数分解

**定義**:
マーケティング目標を具体的な数値指標に分解し、施策との不一致を防ぐ。

**構成要素**:
1. **短期・中長期の売上/顧客数目標**: いつまでにいくら、何件
2. **必要な商談数・リード数**: 逆算した中間指標
3. **ブランド状態**: 認知・評価の目標状態
4. **因数分解式**: 売上 = リード数 × 商談化率 × 受注率 × 顧客単価

**適用方法**:
- 経営課題から逆算して目標設定
- 各KPIを因数分解
- 施策ごとの特性と目標の整合性を確認
- オウンドメディアは成果まで半年〜1年、月間リード上限あり → 2ヶ月で1,000件は不適

### 顧客の検討段階別マーケティング施策マップ

**定義**:
顧客の検討段階(潜在層・準顕在層・顕在層・明確層)に応じた施策選定。

**構成要素**:
1. **潜在層**: カンファレンス、広報、調査リリース
2. **準顕在層**: ブログメディア、ホワイトペーパー、メール配信、セミナー
3. **顕在層**: SEO、リスティング広告、サイト改善
4. **明確層**: 導入事例、ディスプレイ広告、Webサイト改善

**適用方法**:
- Facebook広告 → 潜在層・準顕在層向き(タイムライン閲覧目的)
- SEM(SEO/リスティング) → 顕在層・明確層向き(能動的な情報探索)
- 顧客の情報行動をインタビューで把握(FAX・紙文化の業界も存在)

### 学習曲線と死の谷

**定義**:
マーケティング施策の初期は成果が出ないが、学習を継続することで成果が急上昇する現象。

**構成要素**:
1. **死の谷**: 初期の成果が出ない心理的・数値的ストレス期間
2. **学習時間**: トライの回数と知見・データの蓄積
3. **ブレイクポイント**: 一定の学習後、急激に成果が出る転換点

**乗り越え方**:
1. **意思決定者のコミット**: 経営層が定例会議出席、進捗モニタリング、予算追加判断
2. **専任マーケター配置**: 兼務では死の谷が2〜3倍に延びる
3. **外部知見導入**: 意思決定者+現場マーケター+外部専門家の三位一体
4. **顧客理解の解像度向上**: 会議室ではなく顧客に会う
5. **全施策でROI合わせない**: マイクロソフト実験で1万件A/Bテスト → 成功1/3、無意味1/3、逆効果1/3

**前提条件/注意点**:
打ち手の数が多い会社ほど知見とデータが蓄積し、雪だるま式に施策確度が上がる。

### やらないことリスト(戦略=戦いを略すこと)

**定義**:
成果が出ないことにリソースを使わず、成果が出ることに集中する。

**BtoBで避けるべき施策**:
1. **頻繁なWebサイトリニューアル・A/Bテスト**:
   - BtoBは月数千〜数万セッション → CV率2%→2.1%でもCV数5件増のみ
   - CV率2%達成後は改善停止、コンテンツ作成にシフト

2. **リード数少ない中でのMAツール導入**:
   - 発動条件: 1万件以上のリード、月1本以上のコンテンツ、インサイドセールス体制
   - 1,000件リードで開封率20%×クリック率5% = 再訪10件のみ

3. **休眠リードへの過度な期待**:
   - 休眠理由: プロダクト魅力不足、メッセージ課題、プライシング課題、商談相手ミス
   - 取り組むべきは掘り起こしではなく根本課題の解決

**していることリスト**:
- 顧客に会う(営業ロープレ、商談同行、自ら営業、ユーザーインタビュー)
- カスタマーサポート実施
- 問い合わせ・チャット閲覧
- 展示会呼び込み・説明
- ユーザー会・勉強会開催

### CVポイントの階段設計

**定義**:
見込み顧客が楽に階段を上れるよう、段階的なコンバージョンポイントを設計する最強戦略。

**成功事例: 船井総研**
- **階段構造**: メルマガ → 業界レポートDL → 経営セミナー(1〜3万円) → 経営研究会(年間数十万円) → 月次・PJ型コンサル
- **効果**: 各段階でキャッシュポイント確保、最終CVまでなめらかな誘導

**BtoB SaaS企業の失敗パターン**:
- CVポイントに「無料トライアル」のみ設置
- BtoB特有の社内説明・稟議プロセスを無視
- **改善策**: 資料請求 → お問い合わせ → 導入相談 → 商談 → 無料トライアル → 本導入
- **結果**: CV率2倍達成

**オウンドメディアの失敗パターン**:
- ブログ記事下で「お問い合わせ」「サービス資料」直接誘導
- **改善策**: お役立ち資料ダウンロード → ホワイトペーパー → セミナー → お問い合わせ
- **結果**: 才流の「DOER NOTE」でCV率5倍向上

**購入ハードルを下げる5つの売り方**:
1. **無料トライアル**: kintone 30日間
2. **返金保証**: fondesk 14日間返金保証
3. **廉価版切り出し**: 電通「TANTEKI」(コピーライター+アートディレクターのみ)
4. **作業代行**: Pardot導入支援をパートナー企業に委託
5. **課金方法変更**: 数百万円初期費用 → 月5万円×5年リース

### LTV至上主義とCAC設計

**定義**:
BtoBではCAC削減に限界があるため、LTV向上によりマーケティング自由度を確保する。

**CAC下限の理論**:
- BtoB特性: 衝動買い少、検討期間長、複数人関与、営業介在
- 理想ケース(CPA15,000円、受注率25%、商談1回): CAC = 140,000円
- Facebook広告(CPA3,000円×100件、商談化率5%、受注率20%): CAC = 400,000円
- 展示会+MA+IS活用モデル: CAC最低500,000円

**Sansan成功モデル**:
- **月次解約率**: 0.66%
- **顧客単価**: 156,000円/月
- **理論LTV**: 約2,364万円
- **CAC**: 220〜450万円
- **好循環**: 高LTV → 広告・営業投資可能 → 認知・ブランド育成 → 顧客増加 → 成長

**LTV向上 = 平均顧客単価 × 継続期間**

**顧客単価向上5手法**:
1. **提供価値向上で値上げ**: Amazonプライム 3,900円→4,900円(25.6%増)
2. **中堅・大手対象化**: freee IPO事業部新設
3. **成果連動値付け**: プロレド・パートナーズ(コスト削減3年成果の1/3が対価、営業利益率45.9%)
4. **アップセル・クロスセル**: Sansan導入・運用支援 月額20〜150万円、Salesforce PardotクロスセL
5. **松竹梅プラン**: 極端性の回避、2:5:3の比率で売れる

**継続期間延長6手法**:
1. **オンボーディング**: 無料トライアル後の電話・メール設定サポート
2. **長期契約割引**: Wantedly 6/12/24ヶ月プラン
3. **中堅・大手対象**: SMB月次解約率3〜7% vs 中堅・大手0.5〜1%
4. **低解約セグメント集中**: セグメントA(1%) > B(5%) > C(10%)
5. **業務深入り**: メンバーズEMC(大手向けデジタルマーケ専任チーム)
6. **データ蓄積**: MFクラウド、freee(乗り換えコスト上昇)

**米国SaaS上場企業調査(Blossom Street Ventures)**:
- 年間平均単価150万〜250万円: 上場企業ゼロ(死の谷)
- 上場企業81社中78社: 中堅・大手企業ターゲット
- 2択: 年間平均単価50万円以下(セルフサーブ) or 360万円以上(中堅・大手向け)

## 成功事例

### A社: 3年でリード数30倍&業界2位達成

**背景**:
- 某ITシステム企業、業界4位
- テレアポ中心のアウトバウンドでリスト枯渇
- 業界1位・2位を目指し、新チャネル開拓が経営課題

**施策**:
1. **経営課題連動**: 業界1位・2位目標 → 経営陣の強コミット確保
2. **顧客解像度高いチーム**: 経営者+営業執行役員が定例会議参加
3. **代表顧客10社分析**: 購買プロセス把握 → S社既存システム5年契約縛り判明
4. **競合分析**: 上位2社は顕在層施策充実、潜在層が手薄 → 潜在層に勝機
5. **施策実行**:
   - 顕在層: リスティング広告、SEO、サイトCV率2倍化、導入事例月2本
   - 準顕在層: ブログメディア、ホワイトペーパー、メール月2回、セミナー
   - 潜在層: 年1回カンファレンス、調査リリース広報
   - やらないこと: テレアポ、郵送DM、展示会
6. **定例会議**: 月2回×3年間、KPI進捗確認、追加施策検討、経営者が予算判断

**結果**:
- リード数: 月数十件 → 月1,000件弱(約30倍)
- ポジション: 業界4位 → 業界2位達成
- 副次効果: 1年後にブランド認知拡大、商談化率・受注率向上

### 才流コーポレートサイト: CV率最適化後のリード数2倍化

**背景**:
CV率向上に注力していたが、頭打ち懸念

**施策**:
- リリース後6ヶ月: 高頻度サイト改善
- CV率2%達成時点: 改修停止
- リソースシフト: 認知拡大・集客強化のコンテンツ作成

**結果**:
2ヶ月後にリード数2倍達成
