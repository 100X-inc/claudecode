# LIPS x Braze 実装調査レポート v1

## エグゼクティブサマリー

本レポートは、LIPS（株式会社AppBrew）におけるBraze導入に向けた市場調査・競合分析の結果をまとめたものである。国内外のコスメ・ビューティー業界におけるBraze活用事例、業界ベンチマーク、ベストプラクティスを調査し、LIPS × Brazeの実装戦略に資する知見を整理した。

### 主要ファインディング

1. **@cosme事例の成功**: Braze導入3ヶ月でMAU 150%達成、1年で700施策実行
2. **プッシュプライマーの重要性**: e.l.f.は教育的アプローチで23%オプトイン達成
3. **ゲーミフィケーションの効果**: Sephora SEAは132%の購入率向上を実現
4. **マツキヨ連携の優位性**: オフライン接点×データ統合は競合優位の源泉
5. **Day 30リテンション目標**: 業界平均5%を超える7-10%を目指すべき

## 1. 競合・先行事例分析

### 1.1 アイスタイル（@cosme）- 国内最重要参考事例

#### 概要
日本最大級の美容ポータル（月間1,670万ユーザー）がBrazeを導入し、顧客エンゲージメントを劇的に改善した事例。

#### 導入背景と課題
- メディア、EC、店舗の多チャネル展開によるユーザー分断
- 全ユーザーへの一律情報配信
- パーソナライゼーションの欠如

#### 戦略: ゴールデンルート
- アプリを全チャネルのハブとして位置付け
- WEB/アプリはSDK、店舗/ECはAPIで連携
- 「ユーザーアクションをトリガーとしたCRM」の実現

#### 成果

| 指標 | 結果 | 期間 |
|------|------|------|
| 施策実行数 | 700本 | 1年間 |
| 月間アプリDL数 | +210% | - |
| 特定ユーザー購買率 | +200% | - |
| アプリMAU | +150% | 導入3ヶ月 |
| 購買UU | +110% | - |

#### LIPSへの示唆
- **マルチチャネル統合**: マツキヨ連携で同様のアプローチ可能
- **高速PDCA**: 1日2施策ペースの実行体制構築
- **セグメント配信**: 流入チャネル別の情報出し分け

### 1.2 Sephora SEA - ゲーミフィケーション先進事例

#### Ang Paoキャンペーン（旧正月）
- バーチャル紅包配布による文化的関連性のあるエンゲージメント
- **成果: 購入率132%向上**
- **重要発見**: 低価値景品の受取者が最もコンバージョン（ゲーム性自体が価値）

#### Virtual Artist AR機能促進
- ターゲット: 30日以内にメイク商品閲覧したがAR未使用のユーザー
- 3チャネル（プッシュ、Content Cards、IAM）で動画配信
- **成果: AR採用率28%増、トラフィック48%増**

#### LIPSへの示唆
- **文化的イベント活用**: 季節キャンペーンのゲーミフィケーション
- **行動ベースターゲティング**: 機能未使用者への啓発
- **企画→実行の高速化**: 1ヶ月で完結

### 1.3 e.l.f. Beauty - ロイヤリティ×Braze統合の最新事例

#### 概要
$10億ブランドのe.l.f.がBrazeで達成した最先端のエンゲージメント戦略。

#### プッシュプライマー戦略
- プッシュ通知の価値を事前教育
- **成果: オプトイン率最大23%、プッシュ有効ユーザーは日次セッション500%増**

#### Beauty Squad Replay（年間レビュー）
- 28カスタムIAMスクリーン + 21 Content Cards
- 高度にパーソナライズされた体験
- **成果: ソーシャルシェア率25%**

#### オペレーション効率化
- テンプレート: 100個→6個に削減
- 対応商品ライン: 4→16に拡大

#### 総合成果

| 指標 | 結果 |
|------|------|
| 月間アプリ使用量 | +125% |
| ロイヤリティオファー交換 | +58% |
| キャンペーンエンゲージメント | +102% |

#### LIPSへの示唆
- **プッシュプライマー必須**: iOS成功の鍵
- **年間レビュー施策**: パーソナライズ×シェアの好循環
- **テンプレート効率化**: スケーラビリティ確保

### 1.4 Ulta Beauty - ロイヤリティプログラムの極致

#### ロイヤリティ規模
- 会員数: 4,440万人
- **売上の95%がロイヤリティ会員から**

#### アプリ活用
- EC売上の67%がアプリ経由
- 「店舗でのショッピングコンパニオン」としての位置付け

#### テクノロジー成果（Adobe CDP）
- メールクリック率: +316%
- コンバージョン率: +34%

#### LIPSへの示唆
- **ロイヤリティの重要性**: 売上の大半が会員から
- **フィジタル体験**: 店舗×アプリの融合

## 2. Brazeベストプラクティス

### 2.1 Canvas設計の基本フレームワーク

#### 3つの問い
1. **What**: ユーザーに何を達成してもらいたいか
2. **Who**: リーチしようとしているユーザーは誰か
3. **Why**: なぜこのキャンバスが必要か

#### コンポーネント活用
- **条件分岐**: セグメント別ワークフロー
- **遅延ステップ**: 適切な間隔設定
- **Content Cards**: アプリ内プロモーション
- **実験パス**: ABテストの組み込み

### 2.2 プッシュ通知最適化

#### プッシュプライマー戦略（最重要）
- iOS: 1回の許可要求でユーザーの決定が最終
- **事前教育IAM**でオプトイン率を劇的に向上
- タイミング: 価値あるアクション完了後（例: 注文後の追跡情報提供時）

#### コンテンツガイドライン
- タイトル・本文: 各30-40文字
- プッシュの価値と内容を具体的に説明
- アプリ内で購読管理を提供

#### プラットフォーム差異
- Android: 通知保持、高エンゲージメント傾向
- iOS: 1通知開封で他はクリア

### 2.3 2025年リテールトレンド

| トレンド | 内容 |
|---------|------|
| フィジタル体験 | オンライン/オフライン統合 |
| データドリブン | リアルタイムパーソナライゼーション |
| 感情的つながり | 91%のリテーラーがテクノロジー活用 |
| 3チャネル以上 | トップパフォーマーは16%多く活用 |
| AI活用 | 39%がデータ分析に活用 |

## 4. LIPS × Braze 実装戦略提言

### 4.1 フェーズ1: 基盤構築（0-3ヶ月）

#### 優先施策
1. **プッシュプライマー実装**
   - 目標: オプトイン率60%以上
   - 手法: 価値訴求IAM、ベストタイミング特定

2. **基本セグメント設計**
   - 流入チャネル別（オーガニック/広告/マツキヨ連携）
   - 行動ベース（閲覧カテゴリ、投稿有無、購買履歴）
   - 属性ベース（年代、肌質、パーソナルカラー）

3. **ウェルカムジャーニー構築**
   - Canvas: 教育→価値提示→初回アクション促進
   - 複数チャネル活用（プッシュ、IAM、メール）

### 4.2 フェーズ2: 最適化（3-6ヶ月）

#### 優先施策
1. **ABテスト文化の定着**
   - 全施策でコントロールグループ設定
   - 配信タイミング、コピー、クリエイティブの検証

2. **マツキヨ連携ジャーニー**
   - 店舗来店促進キャンペーン
   - オフライン購買データの統合
   - 「LIPSで見た商品をマツキヨで」のクロスセル

3. **パーソナライゼーション強化**
   - レコメンド精度向上
   - Content Cardsによるダイナミックコンテンツ

### 4.3 フェーズ3: スケール（6-12ヶ月）

#### 優先施策
1. **ゲーミフィケーション施策**
   - 季節イベント（正月、バレンタイン、コスメの日等）
   - スカベンジャーハント型キャンペーン
   - 口コミ投稿チャレンジ

2. **ロイヤリティプログラム統合**
   - マツキヨポイントとの連携
   - ティア制の導入検討

3. **AI活用**
   - 最適配信タイミングの自動化
   - 顧客感情分析
   - 離脱予測と防止施策
