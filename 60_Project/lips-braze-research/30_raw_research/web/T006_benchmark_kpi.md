## モバイルアプリ リテンションベンチマーク（2025-2026年）

### 業種別 Day 30 リテンション率

| 業種 | リテンション率 |
|------|---------------|
| ニュース | 11.3% |
| ビジネス | 5.1% |
| ショッピング | **5.0%** |
| 金融 | 4.6% |
| フード＆ドリンク | 3.7% |
| ヘルス＆フィットネス | 3.7% |

### ショッピング/マーケットプレイスアプリの特徴
- 購買サイクルが不規則のため予測困難
- リテンションは以下に依存:
  - 購買サイクル
  - 季節トレンド
  - パーソナライズドレコメンデーション
- ホリデーシーズン、バック・トゥ・スクール期にスパイク

### ファッション・ビューティーアプリの傾向
- 購入していなくても頻繁にブラウジング
- カジュアルブラウジングが購入間のエンゲージメント維持に貢献
- パーソナライゼーションがゲームチェンジャー
- スタイル、サイズ、予算の学習によるレコメンデーション

## DAU/MAU比率（スティッキネス）ベンチマーク

### 一般的な目安

| 評価 | DAU/MAU比率 |
|------|-------------|
| 優秀 | 40%以上 |
| 低い | 20%以下 |
| B2B SaaS適正値 | 10-20% |
| B2C SaaS適正値 | 30-50% |
| 高エンゲージメント | 50%超 |

### 業種別事例

| アプリ/業種 | DAU/MAU比率 |
|------------|-------------|
| WhatsApp | 約70% |
| LinkedIn | 約20% |
| Gunosy（キュレーション） | 46%（2014年） |
| ニュースアプリ/SNS | 50-75% |
| 小売/ECアプリ | 低め（来店頻度依存） |

### コスメ/ビューティーアプリの考慮事項
- 小売/ECに近い特性（来店頻度が低い商材）
- 日次起動のきっかけ作りが課題
- コンテンツ（口コミ、トレンド）で日次アクセスを促進可能

## ビューティーブランド向け主要KPI

### エンゲージメント指標
- クライアントリテンション率
- 平均チケット価値
- サービスあたりリテール売上
- 顧客満足度スコア

### 収益性指標
- **LTV/CAC比率**: 3:1以下は改善が必要
- **コンバージョン率**: 5-7%超が効率的

### Ulta Beautyの成果（参考）
- メールクリック率: **316%増**
- コンバージョン率: **34%向上**

## リテンション改善のベストプラクティス

### 1. パーソナライゼーション
- AIパワードインサイトでユーザー行動パターン追跡
- アプリ機能の微調整
- 嗜好学習によるレコメンデーション

### 2. ゲーミフィケーション
- リワード
- チャレンジ
- 進捗トラッキング

### 3. プッシュ通知
- 戦略的タイミングが重要
- 不適切な実行はアンインストールを誘発

### 4. ロイヤリティプログラム
- リワードシステムがユーザーライフタイムバリューを延長

## LIPSへの示唆

### 目標設定の参考値

| 指標 | 参考値 | 備考 |
|------|--------|------|
| Day 30 リテンション | 5-7% | ショッピングアプリ平均超え |
| DAU/MAU比率 | 20-30% | コンテンツ力で上振れ可能 |
| LTV/CAC比率 | 3:1以上 | 必須ライン |
| コンバージョン率 | 5-7%超 | EC効率化目標 |

### 改善レバー
1. **コンテンツドリブン**: 口コミ・トレンドで日次アクセス促進
2. **パーソナライゼーション**: 肌質・好みに基づくレコメンド
3. **ゲーミフィケーション**: ポイント・チャレンジ・進捗
4. **マツキヨ連携**: オフライン接点でリテンション向上
