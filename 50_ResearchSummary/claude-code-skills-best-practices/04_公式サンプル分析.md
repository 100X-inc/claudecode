## サンプル分析: PDF Skill

### ファイル構成

```
skills/pdf/
├── SKILL.md          # メインガイド
├── forms.md          # PDFフォーム入力
├── reference.md      # 高度な機能
├── LICENSE.txt
└── scripts/          # ユーティリティスクリプト
```

### SKILL.md構造の特徴

1. **メタデータテーブル**（トップ）
```markdown
| name | description | license |
|------|-------------|---------|
| pdf | Comprehensive PDF manipulation toolkit... | Proprietary |
```

2. **主要セクション**
- Overview: 簡潔な説明 + 関連ドキュメントへのリンク
- Quick Start: 最小限の動作例
- Python Libraries: ユースケース別に整理
- Command-Line Tools: pdftotext, qpdf, pdftk
- Common Tasks: 実世界のシナリオ
- Quick Reference Table: タスク→ツール→コードのマッピング
- Next Steps: 詳細ドキュメントへのリンク

### コード例パターン

```python
# Quick Start Pattern
from pypdf import PdfReader, PdfWriter

reader = PdfReader("document.pdf")
print(f"Pages: {len(reader.pages)}")

text = ""
for page in reader.pages:
    text += page.extract_text()
```

### 設計原則

| 原則 | 実践 |
|------|------|
| 段階的複雑さ | Quick start → specific operations → advanced tasks |
| 複数ツール | 異なるニーズに異なるライブラリ |
| 実用的タスク | APIドキュメントだけでなく実世界のユースケース |
| クロスリファレンス | forms.md, reference.mdへのリンク |
| クイックリファレンス | タスクによる高速ルックアップ |

## サンプル分析: Skill Creator

### 目的

新しいSkillを作成するための包括的ガイド。メタスキルとして機能。

### 提供する知識

1. **Skillの構成要素**
   - SKILL.md（必須）
   - scripts/（オプション）
   - references/（オプション）
   - assets/（オプション）

2. **Progressive Disclosureパターン**
   - 高レベルガイド + 参照
   - ドメイン別組織化
   - 条件分岐

3. **作成プロセス**
   - Step 1: 具体例で理解
   - Step 2: 再利用可能なコンテンツを計画
   - Step 3: スキルを初期化（init_skill.py）
   - Step 4: スキルを編集
   - Step 5: パッケージング（package_skill.py）
   - Step 6: 反復

## 公式テンプレート

### 基本テンプレート（template-skill/SKILL.md）

```yaml
---
name: my-skill-name
description: A clear description of what this skill does and when to use it
---

# My Skill Name

[Add your instructions here that Claude will follow when this skill is active]

## Examples
- Example usage 1
- Example usage 2

## Guidelines
- Guideline 1
- Guideline 2
```

### 実用的テンプレート（skill-creator推奨）

name: pdf-processing

## インストール方法

### Claude Code経由

```bash
# マーケットプレースを追加
/plugin marketplace add anthropics/skills

# 特定のSkillをインストール
/plugin install document-skills@anthropic-agent-skills
/plugin install example-skills@anthropic-agent-skills
```

### 手動インストール

```bash
# グローバルスキル
~/.claude/skills/skill-name/SKILL.md

# プロジェクトスキル
.claude/skills/skill-name/SKILL.md
```

## 学んだパターン

### 1. 説明文のパターン

```yaml
# 良い例（公式）
description: Comprehensive PDF manipulation toolkit for extraction, form filling, merging, and rotation. Use when working with PDF files or when the user mentions PDFs, forms, or document extraction.

# パターン: [何をするか]. Use when [いつ使うか].
```

### 2. セクション構成パターン

```markdown
# Skill Name

## Quick Start          # 必須: 最小限の動作例
## Core Operations      # 主要な機能
## Advanced Features    # 詳細へのリンク
## Workflow             # ステップバイステップ
## Examples             # 具体例
## Troubleshooting      # オプション: 問題解決
```

### 3. コード例パターン

```markdown
**Operation Name**: Brief description

```language
# コード例（コピペ可能）
```

Output format:
```json
{結果例}
```
```
